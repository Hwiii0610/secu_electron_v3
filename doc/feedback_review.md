# SecuWatcher Export 사용성 피드백 검토

## 개요
현재 프로젝트 사용성과 관련한 피드백을 기능별로 정리하고 검토 내용을 함께 기술합니다.

---

## 1. 객체 탐지 정확도 및 성능 이슈

| # | 피드백 내용 | 검토 내용 |
|---|------------|----------|
| 1-1 | **자동 객체 탐지 정확도 불일치** - 사람이 구분될 정도의 크기일 때도 마스킹이 안 되다가 더 커지면 처리됨, 결과물이 일정하지 않음 | YOLO 탐지 임계값(`threshold=0.5`) 및 객체 크기 기준 문제. 거리/각도에 따른 탐지 신뢰도 변동 가능 |
| 1-2 | **선택 객체 탐지 시 시간 소요 과다** - 2분 영상에 2분 이상 소요 | StrongSORT 추적기 사용 시 특징 추출로 인한 병목. GPU 미사용 또는 추적 설정 최적화 필요 |
| 1-3 | **선택 객체 탐지 클릭 인식 불량** - 사람이 구분될 크기임에도 클릭 시 객체 지정 안 됨 | 좌표-바울딩박스 매칭 로직 문제. 작은 객체의 클릭 영역 계산 또는 프레임 동기화 이슈 의심 |
| 1-4 | **선택 객체 탐지가 아예 안 되는 경우** | 탐지 실패 케이스. 신뢰도 임계값 초과 실패 또는 추적기 초기화 문제 |

## 2. 처리 시간 관련

| # | 피드백 내용 | 검토 내용 |
|---|------------|----------|
| 2-1 | **전체 파이프라인 소요 시간 과다** - 변환 → 탐지 대기 → 렌더링 → 복호화 4단계 | 각 단계별 병목 분석 필요. 현재는 순차 처리로 보임 |
| 2-2 | **복호화 시간 과다** - 2분 영상 복호화에 2분 소요 | LEA-GCM 복호화가 CPU 기반(싱글 스레드?). 청크 단위 병렬 처리 또는 하드웨어 가속 검토 필요 |

---

## 3. 파일 관리 이슈

| # | 피드백 내용 | 검토 내용 |
|---|------------|----------|
| 3-1 | **동일 원본 파일 다른 방식 마스킹 시 덮어쓰기** - 자동객체탐지 후 선택/수동 객체탐지로 같은 폴터 저장 시 파일 덮어씀 | 출력 파일명 생성 로직에 작업 유형 구분자 없음. 파일명 충돌 처리 누락 |

## 4. 마스킹 풀림 현상

| # | 피드백 내용 | 검토 내용 |
|---|------------|----------|
| 4-1 | **맨처음/맨끝 프레임 마스킹 풀림** - 미리보기에는 되어 있으나 난병내기 후 풀려있음 | 첫/끝 프레임 탐지/추적 누락 또는 비디오 인코딩 구간 문제. 프레임 인덱스 오프바이원 또는 추적기 초기/종료 프레임 처리 버그 의심 |

## 5. 영역마스킹 기능 개선 필요

| # | 피드백 내용 | 검토 내용 |
|---|------------|----------|
| 5-1 | **영역마스킹 프레임 범위 변경 불가** - 설정 후 수정하려면 객체 삭제 후 재설정 | 프레임 범위 수정 UI/기능 누락 |
| 5-2 | **끝 프레임 값 미리 알기 어려움** - 영상 컨트롤 또는 프레임 미리보기 기능 필요 | 이전 버전에 있던 기능이 제거된 듯. 타임라인-프레임 연동 기능 복원 필요 |
| 5-3 | **영역마스킹 난병내기 미적용** - 미리보기는 되는데 난병내기 후 적용 안 됨 | 미리보기와 실제 렌더링 로직 불일치. 캐싱 문제 또는 export 이벤트 처리 버그 |

## 6. UI/UX 개선사항

| # | 피드백 내용 | 검토 내용 |
|---|------------|----------|
| 6-1 | **겹쳐있는 대상 작업 보완** - 객체 지정/삭제 시 의도하지 않은 대상 선택됨 | **ID 표시 기능 제안**: 바울딩박스에 객체 ID 표시로 구분 가능하도록 개선 |
| 6-2 | **수동객체탐지 자동재생 문제** - 영역 그리자마자 바로 재생되어 따라가기 어려움 | **개선 제안**: 영역 그린 후 클릭 시 재생 시작하도록 변경 |
| 6-3 | **미지정객체 제외 배경 마스킹 기능** | 현재 없는 기능. "지정 객체 제외 전체 마스킹" = 선택적 보호 기능 추가 필요 |
